# AITalkbotproject

# 개요
설계 목표<br>
이미지 예측을 통해 랜드마크 정보 제공<br>
랜드마크 주변의 추천 장소 서비스<br>
챗봇 기능으로 한류 여행의 호기심 자극<br>
# 사용 기술
-Image Detection<br>
-Keras CNN Model <br>
-Saltlux ChatBot <br>
-Flask<br>
# 사용 언어
Python<br>
